# WHYN_Spring


#### 기존 MVC로 작업한 Whyn Web을 Spring 구조로 변경 하였습니다.

이 프로젝트에서는 admin 페이지가 주된 페이지 였기때문에 기존에 있던 사용자 페이지는 구현하지 않고 admin에 집중하였습니다.
영상에는 사용자페이지도 같이 나와있지만 admin페이지는 spring과 같습니다.
